Quick Start Guide

1. Select a Protocol: Choose an Acid-Base or Redox lab.
2. Set Molarity: Check the Molarity Guide before entering.
3. Choose Indicator: Select the correct color indicator (or Starch for Redox).
4. Titrate: Add titrant (+1.0mL) initially, then slow to 💧Drop (0.02mL) near the endpoint.
5. Analyze: Mark your equivalence point and complete the Post-Lab report.
